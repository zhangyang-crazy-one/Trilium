# /next - 获取下一步建议

作为任务规划助手，请分析当前项目状态，给出下一步开发建议。

## 你需要做的：

1. 读取 docs/TODO.md 了解待办任务
2. 读取 docs/PROJECT_STATUS.md 了解进度
3. 扫描代码中的 TODO/FIXME 注释

## Git 可用时（加分项）：
- 查看最近 5 条 Git 提交了解当前工作

## Git 不可用时（降级方案）：
- 基于 TODO 列表提供建议

## 输出格式：

# 🎯 下一步建议

## 当前状态
- 整体进度：XX%
- 待办任务：X 个（高优先级 Y 个）

## 🚀 建议优先级

### 🔴 高优先级
1. 任务1
   - 原因：XXX
   - 预计耗时：X 小时
   - 位置：XXX

### 🟡 中优先级
1. 任务2

### 🟢 低优先级
1. 任务3

## 具体行动建议
1. 优先处理高优先级任务
2. 参考 docs/PROJECT_STATUS.md 了解详情
3. 完成后运行 `/update-status` 更新状态
